Get Account Scheduled Payments

GET/accounts/{accountId}/scheduled-payments

Returns the scheduled payments of the specified account.

Path Parameters​

NameTypeRequiredDescription
accountIdstringtrueID of the account.

Headers​

NameTypeRequiredDescription
AuthorizationstringtrueBearer followed by a base64 encoded OAuth access token.
X-CorAPI-Client-IDstringtrueID of the client forwarded to the provider (SCOPE: FI).
X-Correlation-IDstringtrueUnique ID (defined by the caller) which will be reflected back in the response.
User-AgentstringtrueName and version of the of the Client software.
X-CorAPI-Target-IDstringfalseID of the target, e.g., a financial institution (SCOPE: FI - optional).
X-PSU-IP-AddressstringfalseIP address of the user initiating the operation (SCOPE: FI - optional).
X-PSU-User-AgentstringfalseUser of the client software (SCOPE: FI - optional).

200 OK​

Returns the scheduled payments of the specified account.

400 Bad Request​

The format of the request was invalid.

401 Unauthorized​

The request has not been applied because it provides no valid authentication credentials for the target resource.

403 Forbidden​

A valid OAuth Token was received, but access was denied.

404 Not Found​

Either the endpoint does not exist or a requested resource is not yet available.

405 Method Not Allowed​

The method received in the request-line is known by the origin server but not supported by the target resource.

500 Internal Server Error​

The server encountered an unexpected condition that prevented it from fulfilling the request.

501 Not Implemented​

The server does not support the functionality required to fulfill the request.

503 Service Unavailable​

The server is currently unable to handle the request due to a temporary overload or scheduled maintenance.
